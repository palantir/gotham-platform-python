# Permission

A permission, one of READ, WRITE, or OWNER. Each successive permission implies the previous ones, so WRITE
implies READ, and OWNER implies READ and WRITE.


| **Value** |
| --------- |
| `"READ"` |
| `"WRITE"` |
| `"OWNER"` |
